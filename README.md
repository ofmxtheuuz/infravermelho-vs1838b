  <li>LED Difuso 5mm Vermelho (1 unidade)</li>
  <li>LED Difuso 5mm Verde (1 unidade)</li>
  <li>LED Difuso 5MM Amarelo (1 unidade)</li>
  <li>Resistor de 150Ω (3 unidades)</li>

  <li>Cabos Jumper macho-macho (9 unidades)</li>
